# OSBE Markdown Clipper Privacy Policy

OSBE Markdown Clipper converts the current page or selected content to Markdown only when the user invokes it from the popup or context menu.

## Data handled

The extension temporarily reads the active page or selection, page metadata, and referenced images needed to create the requested Markdown output. Clipping preferences are stored locally in `chrome.storage.local`.

## Data sharing

Page content, selections, generated Markdown, images, and preferences are not transmitted to OSBE or sold or shared for advertising. Referenced images may be fetched directly from their source websites when the user asks to include local image assets.

## User control

The extension uses `activeTab`, so page access is granted only after the user invokes the extension. Users can clear saved preferences by removing the extension or clearing its extension storage.

## Remote code

The extension does not execute remote code.
